# javascriptMask
Projeto javascript simples para dar comportamento de maskedit às caixas de texto.

Contempla os seguintes formatos prontos (Pt-br):
  • CPF
  • CNPJ
  • DDD
  • Telefone (com e sem o nono dígito)
  • Celular
